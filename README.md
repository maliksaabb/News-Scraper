# News-Scraper
## Description

News-Scraper is a web app that lets users view and save  news scraped from the news website


## Tech Used
- Handlebars
- JavaScript
- jQuery
- Node
- Express
- MongoDB
- Mongoose
- Cheerio
- Axios
- Heroku
